# Monadic
El repositorio esta compuesto de 2 archivos uno representa el back-end  (BACK) y el otro front-end (FRONT)

## Empezando BACK

```bash
# Clonar repo
git clone https://github.com/NoemiFk/Monadic.git
 
# Cambiar directorio
cd BACK

# Instalar dependencias NPM
npm install

# Inicie el servidor
nodemon
# o 
nodo app.js

```

## Empezando FRONT

```bash
 
# Cambiar directorio
cd FRONT

# Instalar dependencias NPM
npm install

# Inicie el servidor
npm start


```
